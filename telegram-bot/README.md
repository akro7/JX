# AKRO GOXI — Telegram Bot

بوت تليجرام بيتكلم مع سيرفر الموقع (akrogoxi.x10.mx) عشان يولّد ويفحص الـ Keys من غير ما تفتح المتصفح.

## الخطوات

### 1) اعمل بوت على تليجرام
كلم **@BotFather** → `/newbot` → هياديك **Token**.

### 2) خد سر البوت من الموقع
افتح ملف `.env` في السيرفر وغيّر السطر:
```
bot.secret = 'CHANGE_ME_TO_A_LONG_RANDOM_SECRET'
```
لسر عشوائي طويل بتختاره انت (استخدمه بنفس القيمة في الخطوة الجاية).
⚠️ لازم تتأكد إن x10.mx فعلاً بيقرا متغيرات `.env` بتاعة CodeIgniter، لو لأ حط القيمة مباشرة جوه `app/Config/Bot.php` بدل الـ env (قولّي لو حصل كده وأظبطهالك).

### 3) ضيف الأسرار دي في إعدادات الريبو على جيتهاب
Settings → Secrets and variables → Actions → New repository secret:

| الاسم | القيمة |
|---|---|
| `TELEGRAM_BOT_TOKEN` | التوكن من BotFather |
| `SITE_API_URL` | `https://akrogoxi.x10.mx` |
| `BOT_API_SECRET` | نفس القيمة اللي حطيتها في `.env` |

### 4) فعّل الـ Workflow
هيلاقي نفسه شغال أوتوماتيك كل 5 دقايق (الـ cron جوه `.github/workflows/telegram-bot.yml`)، أو شغّله يدوي من تاب **Actions → AKRO GOXI Telegram Bot → Run workflow**.

## استخدام البوت
البوت دلوقتي شغال بقايمة أزرار زي الموقع بالظبط — مفيش أوامر تكتبها:

1. ابعت `/start`
2. دوس **🔑 Generate License**
3. أول مرة هيطلب منك اليوزر بتاعك في الموقع (بيتذكره بعد كده)
4. اختار اللعبة (PUBG / 8 Ball Pool) من الأزرار
5. اختار المدة من الأزرار
6. اختار عدد الأجهزة (أو دوس "رقم تاني" واكتبه بنفسك)
7. تأكيد → البوت يولّد الـ Key ويبعتهولك

فيه كمان زرار **📊 Check Key Status** و **💰 Check Balance** من نفس القايمة الرئيسية.

## ليه شغال بالشكل ده؟
GitHub Actions مش بيسمح بعملية شغالة 24/7 مجانًا، فالبوت بيشتغل polling لمدة ~4.5 دقيقة كل مرة يتشغل، والـ workflow بيعيد تشغيل نفسه كل 5 دقايق. مفيش تأخير محسوس في الاستخدام العادي، بس مش استجابة فورية 100% زي بوت شغال بشكل مستمر على سيرفر حقيقي.
